# TinyApp Project

Shoppy is a full stack application using React and Firebase that allows users to shop clothes and admin to add products.

## Dependencies

- React
- React-dom
- React-router-dom
- React-icons
- React-queriy
- Uuid
- Tailwind
- Firebase

## Getting Started

- Install all dependencies (using the `npm install` command).
- And you can run `npm start`
